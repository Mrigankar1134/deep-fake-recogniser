# DeepFake Detection using VGG16 and SVM

This project focuses on detecting deepfake (synthetic) images using a combination of fine-tuned VGG16 deep learning model and an SVM classifier. It includes a training script (`train.ipynb`) and a Streamlit web application (`app.py`) to test the model on real and fake face images.

## рџ”Ќ Overview

- **Model**: Pretrained VGG16 from PyTorch, fine-tuned for feature extraction.
- **Classifier**: Support Vector Machine (SVM) for final classification based on extracted features.
- **Web Interface**: Streamlit app to upload an image and predict whether it's **Real** or **Fake**.
- **Dataset**: 140K Real and Fake Faces from Kaggle

## рџ§  Training the Model

The training process is implemented in `train.ipynb`:
1. Load the dataset and apply image transformations.
2. Fine-tune the **VGG16** model (pretrained on ImageNet).
3. Extract features using VGG16.
4. Train an **SVM** classifier using these features.
5. Save the final model to `model_15.pth`.
